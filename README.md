# AnaBoard ‚Äì Keyboard by Analysa

*AnaBoard* is a privacy-focused, customizable, open-source Android keyboard based on
*HeliBoard / OpenBoard*, which are derived from the AOSP Keyboard (LatinIME).

AnaBoard provides a smooth typing experience and also includes
*optional smart writing tools powered by Analysa services*.

## üìå Features

### Core Keyboard Features

- Dictionary-based suggestions and spell check
  - You can add your own dictionaries or use
    https://codeberg.org/Helium314/aosp-dictionaries
- Customize themes (colors, backgrounds)
  - System day/night support (Android 10+)
  - Dynamic colors (Android 12+)
- Custom keyboard layouts
  - See [layouts.md](layouts.md)
- Multilingual typing
- Emoji and symbol suggestions
- Clipboard history
- One-handed mode
- Split keyboard
- Number pad
- Backup & restore settings and learned words
- Glide typing support (via external library)

## üîê Privacy

AnaBoard is designed with privacy in mind:

- Password and sensitive fields are *never processed*
- Only text you select for a feature is sent to the server
- No background keylogging or tracking
- No ads or analytics SDKs
